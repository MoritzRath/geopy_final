# geopy_final

Hi! 

This work was done as a final assignment for the graduate course "Geoprocessing with Python" at the Geographic Institute at Humboldt-University of Berlin in the winter semester 2025/26. It features analysis of Ice Velocity on Vega Island, Antarctica, from 2020 - 2024 using Sentinel-1 Radar Imagery. 

The preprocessed S1 data is derived from GEE, as well as a Glacier Inventory and a DEM. Additional geodata, that wasn't derived from GEE can be found in the folder "geodata" (e.g. outlines for a ROI & downloaded data from GEE).

Feel free to adjust your path and reproduce my work or expand it to other time frames. If you find any incoherencies, hic-ups or  if you're interested in the resulting report - feel free to reach out to me!
